# Backyard Upgrades Publish-Ready Package

This bundle is meant to hand directly to Codex or a developer working on the Backyard Upgrades static site.

## What is inside

- `content/articles/` — 10 rewritten article markdown files
- `content/legal/` — about, disclosure, privacy, and contact copy drafts
- `data/articles.json` — enriched article metadata
- `data/article_product_map.csv` — product recommendations and affiliate URLs
- `docs/PUBLISH_READY_HANDOFF.md` — implementation instructions
- `docs/LAUNCH_CHECKLIST.md` — publish QA checklist

## Notes

- Amazon affiliate tag used: `simplekit-20`
- Some URLs are direct product links and some are affiliate-tagged Amazon search links
- Search links should be upgraded to exact product pages during final QA where possible
