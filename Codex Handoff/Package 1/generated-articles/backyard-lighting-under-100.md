# Backyard Lighting Under $100

You do not need a big budget to make a backyard feel safer and more inviting after dark. With the right mix of affordable solar path lights and outdoor string lights, it is possible to improve both function and atmosphere for under $100. The key is choosing lighting that solves a specific problem instead of buying several low-impact products.

## Budget Lighting Priorities

Before buying anything, decide what matters most: visibility, ambiance, or both. If your yard feels dark around steps or walkways, put most of the budget into path lighting. If the space is already safe but feels unfinished, string lights often deliver the biggest visual upgrade per dollar.

## Practical Advice

A simple under-$100 setup often works best:
- Use one set of path lights for safety and structure.
- Add one strand of string lights near the main seating area.

Avoid splitting your budget across too many products. A small, well-lit space feels better than a large yard with scattered dim lights. If you need to stretch your money further, start with path lights this season and add string lights later.
